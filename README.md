### Requirements

To install this game, you need to be on Linux, and need to have `gcc`,  `make` and the `CSFML` library.

### Install

Install this game in 3 steps :

1. Clone the repository :
```bash
git clone https://github.com/nl1x/my_hunter
```

 2. Compile the game :
```bash
make
```

 3. Launch the game :
```
./my_hunter
```

### Usage
```bash
./my_hunter [-h] [--help]
```
